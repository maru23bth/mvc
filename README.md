# mvc

![Symfony logo](https://symfony.com/logos/symfony_black_02.svg)


För att klona repot.

```
git clone https://github.com/maru23bth/mvc
```

Installation.
```
composer install
npm install
npm run build
```

För att komma igång med att köra webbplatsen.
```
php -S localhost:8888 -t public
```
